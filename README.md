# 주요사항 체크리스트

- 고객사에서 종료일(ex. 2026-02-19)을 엑셀 컬럼으로 무조건 넣었다는 가정하에 작업된 내역들이 많습니다.
  이 부분에 대해서 넣는지 안넣는지 확인이 필요하긴할거같습니다. 혹시나 안넣는다면... 수정이 많을거같습니다.
- 고객사에서 전화번호를 엑셀업로드시 넣지않으면 무조건 버그가 날겁니다. (ex. SMS발송실패 등)

- 위 2개를 대응하려면 엑셀은 동적으로 처리하되 필수적인 컬럼항목은 유효성검사를 체크해서 업로드못하게막던지 사전에 최소 2~3개의 고정컬럼은 협의가 필요할 수 있을거같습니다..


# 엑셀사본업로드 및 내용 추가 필요 항목
- 인등번호, 인등위치, 신도번호, 대주, 영가여부, 동참자 등 나머지는 추가내용 확인 필요

# 상단 카드섹션 인원 카운트 변경 필요
- 전체, 한달전, 2주전, 종료

# 엑셀업로드 시 목록 추가 or 초기화하고 새로만들기 필요
- 첨부한 엑셀문서의 인등위치 항목에 표시되는 위치가 7군데임, 다른절은 더많을수도 적을수도 있다고 함
- 예를들면 엑셀문서 7개를 업로드하면 7개 내용이 다 남을수 있어야하고 초기화하고 올릴수 있어야 함

# 문자발송
- 문자 기능은 가능하면 알리고로 제안할 예정 알리고홈페이지 들어가면 연동 가이드 있음
- 절마다 사용하는 문자업체가 제각각이라고 함. 차후 추가 작업이 필요할수 있으나 그떄그떄 추가작업으로 새로 계약 맺을 예정 (50만원 100만원 이라도 돈받고 일하자)
- "솔라피"라는 업체 사용하기로 함 => 솔라피 사이트에 연동가이드 있음

# 문자 자동발송 기능
- 한달전 혹은 2주전 혹은 1주전 오전 오후 몇시 라고 저장하면 파일에 써서 저장하고
- 프로그램 실행시 파일 내용 읽어서 해당 시간에 문자 발송
